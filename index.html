Git Hub Project:
      Create an account on GitHub
      Create a local repository using the terminal
      Create a readme.md file to the project
      Commit changes to the project
      Download GitHub for Mac: GitHub Client or continue using terminal: Git via Terminal Docs
      Push your repository to github
      Create a "hello world" index.html file locally, commit and push.
      Create a "project page" (make 'gh-pages' the default branch)
      Make changes to master: add some content to index.html file, commit/push.
      Merge gh-pages with master branch.
      Create a "my user page" (index for all your projects)
      Create an issue and assign to a team member, e.g. "Fix the padding on main headers."
