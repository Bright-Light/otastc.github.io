Git Hub Project: 1. Create an account on GitHub 2. Create a local repository using the terminal 3. Create a readme.md file to the project 4. Commit changes to the project 5. Download GitHub for Mac: GitHub Client or continue using terminal: Git via Terminal Docs 6. Push your repository to github 7. Create a "hello world" index.html file locally, commit and push. 8. Create a "project page" (make 'gh-pages' the default branch) 9. Make changes to master: add some content to index.html file, commit/push. 10. Merge gh-pages with master branch. 11. Create a "my user page" (index for all your projects) 12. Create an issue and assign to a team member, e.g. "Fix the padding on main headers."
